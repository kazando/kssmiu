Files created from the 6 Euclidean-result logs in kss2phylip-n{100,250,500}-l{1,2}-euc.txt

Data files
----------
- euclidean_improvement_l1.dat / l2.dat: mean relative improvement (%) and standard error
- euclidean_time_l1.dat / l2.dat: mean total running time (s) and standard error
- euclidean_summary.csv: long-form summary table
- euclidean_raw_results.csv: parsed raw results (120 rows per p/n combination total 1080 rows)

Plots
-----
Run, in this directory:
  gnuplot fig1_euclidean_improvement.gp
  gnuplot fig2_euclidean_time.gp

The resulting PDFs are Fig. 1 and Fig. 2. Error bars show standard errors over 20 instances.
